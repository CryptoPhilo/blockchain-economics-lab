### 4.3 재생산 방지 메커니즘

**현재 구현:**
```python

    f for f in md_files
```

**강점:**
- Google Drive 파일 ID 기반 → 파일명 변경에도 안정적

**약점:**
- `processed_files.json` 삭제/손실 시 전체 재처리 위험
